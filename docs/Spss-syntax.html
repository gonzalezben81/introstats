GET DATA
  /TYPE=XLSX
  /FILE='C:\path\to\yourfile.xlsx'
  /SHEET=name 'Sheet1'
  /CELLRANGE=FULL
  /READNAMES=ON.
GET DATA
  /TYPE=TXT
  /FILE='C:\path\to\yourfile.csv'
  /DELCASE=LINE
  /DELIMITERS=","
  /ARRANGEMENT=DELIMITED
  /FIRSTCASE=2
  /VARIABLES=var1 F8.2 var2 F8.2 var3 F8.2.

T-test code

T-TEST GROUPS=gender(1 2)
  /VARIABLES=math_score
  /CRITERIA=CI(.95).

Chi-square code

CROSSTABS 
  /TABLES=variable1 BY variable2
  /STATISTICS=CHISQ
  /CELLS=COUNT EXPECTED.

Pearson Correlation

CORRELATIONS 
  /VARIABLES=var1 var2
  /STATISTICS=DESCRIPTIVES.

Spearman Rank Correlation

NONPAR CORR 
  /VARIABLES=var1 var2
  /PRINT=SPEARMAN.

Partial Correlation

PARTIAL CORR 
  /VARIABLES=var1 var2
  /CONTROLLING=control_var.

Scatterplot in SPSS

GRAPH 
  /SCATTERPLOT(BIVAR)=independent_variable WITH dependent_variable.

Bivariate Regression

REGRESSION 
  /DEPENDENT=dependent_variable
  /METHOD=ENTER independent_variable.
